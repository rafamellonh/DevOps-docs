# 🚀 DevOps Pro Journey

Repositório dedicado à minha jornada de estudos em DevOps, com foco em práticas modernas de automação, containers, cloud e observabilidade.

## 🎯 Objetivo
Consolidar conhecimentos práticos e teóricos em DevOps, aplicando conceitos utilizados no mercado em cenários reais.

## 🧰 Tecnologias abordadas
- Docker
- Kubernetes
- Ansible
- Terraform
- CI/CD
- Cloud (Azure/AWS)
- Observabilidade (Prometheus, Grafana)

## 🧪 Abordagem
- Documentação simples e objetiva
- Laboratórios práticos (hands-on)
- Resolução de problemas reais
- Boas práticas de mercado
